The footer below is a contextual menu of action buttons.

Right-aligned, the primary action button is displayed with 100% opacity to exagerate its importance.

The buttons on this page are not set to do anything, but can be linked to trigger JavaScript events, submit forms, or interact with hidden areas on the page.

Other elements that can be used in place of <button> elements are <a> or <label>. Using a label, it is possible to link the footer with hidden radio/checkbox inputs to represent state of the current view.
